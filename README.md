# 🌾 Campo Vivo — O Equilíbrio entre Tecnologia e Meio Ambiente

> **Projeto desenvolvido para o Concurso Agrinho 2026** > **Categoria:** Programação Front-End  
> **Nível:** 1ª Série do Ensino Médio | Rede Estadual de Ensino do Paraná

---

## 📋 Sobre o Projeto

O **Campo Vivo** é uma plataforma web interativa desenvolvida com o objetivo de conscientizar e simular o impacto positivo do uso de tecnologias sustentáveis no agronegócio moderno. Alinhado com o tema do concurso *“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”*, o site combina design minimalista, dados técnicos e ferramentas interativas voltadas para a imersão e experiência do utilizador.

A principal funcionalidade é uma **Calculadora de Impacto Sustentável**, que estima a economia de energia, a redução de emissões de CO₂ e melhorias operacionais com base na escolha de tecnologias como Energia Solar, Energia Eólica e Biodigestores.

---

## ✨ Funcionalidades Principais

- **Calculadora Interativa:** Processamento em tempo real com lógica em JavaScript para simular cenários agrícolas sustentáveis.
- **Painel de Acessibilidade Computacional:**
  - **Ajuste Dinâmico de Texto:** Um slider interativo que altera as fontes de forma proporcional em todo o site usando unidades relativas (`rem`).
  - **Modo Escuro (Dark Mode):** Alternância completa de cores para conforto visual, incluindo a adaptação de cenários, ilustrações SVG e janelas flutuantes.
  - **Persistência de Dados:** Uso de `localStorage` para que as preferências de acessibilidade do utilizador (tamanho da fonte e tema do site) permaneçam guardadas mesmo após recarregar a página.
- **Seção Interativa de Fontes Bibliográficas:** Um modal flutuante moderno e responsivo que exibe as referências de pesquisa (como EMBRAPA, FAO, MME) ao clique de um botão, preservando a estética do layout principal.

---

## 🛠️ Tecnologias Utilizadas

Em total conformidade com os critérios rigorosos do regulamento do Concurso Agrinho, o projeto foi construído **100% nativo (Vanilla)**, sem a utilização de frameworks, bibliotecas prontas ou carregamentos externos (garantindo que o site funcione perfeitamente de forma offline):

- **HTML5:** Estruturação semântica, acessível e otimizada para motores de busca (SEO).
- **CSS3:** Estilização moderna através de Variáveis CSS (*Custom Properties*), animações de transição suaves, Flexbox e Grid Layout para total responsividade em computadores, tablets e telemóveis.
- **JavaScript (ES6+):** Lógica de manipulação do DOM, cálculos matemáticos, escuta de eventos e persistência local.
- **Vetorização SVG:** Gráficos e cenários escaláveis manipulados dinamicamente via CSS de acordo com o tema selecionado.
